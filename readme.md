# JK Electronics

JK Electronics is a full-stack e-commerce application designed for selling electronic products. It features a robust product catalog, secure user authentication, and integrated payment processing.

## Features

- **Product Management**: Browse and search for various electronic items.
- **User Authentication**: Secure signup and login using JWT and bcrypt.
- **Image Hosting**: Product images managed via ImageKit.
- **Payment Integration**: Seamless checkout experience powered by Razorpay.
- **Email Notifications**: Order confirmations and communications via Resend.
- **Database**: Reliable data storage using MySQL.
